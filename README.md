## 📚 Mini DevHub

 A team knowledge base to add, browse, and share learning resources

### What is it?

Mini DevHub is a simple web app that lets you and your team save and organize learning resources. You can add links or upload files, categorize them, and search through everything easily.

No database needed — everything saves right in your browser.

### Features 

Add resources - Save links or upload PDFs, images, documents
Categorize - Group by HTML, CSS, JavaScript, React, or custom tags
Difficulty - Mark as Beginner, Intermediate, or Advanced
Search - Find resources by title
Delete - Delete resources you no longer need

#### Tech stack

CSS
JavaScript

##### How It Works
You add a resource (title, link or file, category, difficulty)

JavaScript saves it to your browser's storage

The resource appears on the page instantly

Even if you close and reopen the browser, your resources are still there
